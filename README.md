 # Playwright API Automation Framework

This repository contains an API automation framework built using **Playwright** and **TypeScript**.  
It is designed to validate REST APIs with token-based authentication, reusable helpers, and scalable test structure.  
The framework is CI-ready and follows real-world industry practices.
